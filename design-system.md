# Retro Design System - Gym Landing Page

## Brand Identity
**FitZone Gym** - Transform Your Body, Transform Your Life

## Visual Style
- **Theme**: High-contrast retro design
- **Aesthetic**: Vintage-inspired with modern functionality
- **Personality**: Bold, energetic, nostalgic

## Typography Scale

### Font Families
- **Primary**: Macondo (headings and body text)
- **Display**: Macondo (hero titles and emphasis)
- **Monospace**: JetBrains Mono (technical text, numbers)

### Font Weights
- Thin: 100
- Extra Light: 200
- Light: 300
- Normal: 400 (default)
- Medium: 500
- Semi Bold: 600
- Bold: 700
- Extra Bold: 800
- Black: 900

## Color Palette

### Primary Colors
- **Primary**: #3B82F6 (Blue - main brand color)
- **Secondary**: #8B5CF6 (Purple - accent color)

### Semantic Colors
- **Success**: #16A34A (Green - positive actions)
- **Warning**: #D97706 (Orange - caution states)
- **Danger**: #DC2626 (Red - errors, alerts)

### Base Colors
- **Surface**: #FFFFFF (White - backgrounds)
- **Text**: #111827 (Dark gray - primary text)

## Spacing Scale
- **XS**: 4px (tight spacing)
- **SM**: 8px (small gaps)
- **MD**: 12px (medium spacing)
- **LG**: 16px (standard spacing)
- **XL**: 24px (large gaps)
- **2XL**: 32px (section spacing)

## Accessibility Standards
- **Compliance**: WCAG 2.2 AA
- **Focus States**: Visible and high-contrast
- **Keyboard Navigation**: Full support
- **Color Contrast**: Minimum 4.5:1 for normal text
- **Touch Targets**: Minimum 44px for interactive elements

## Component Guidelines

### Buttons
- Use high-contrast colors
- Maintain minimum touch target size
- Provide clear focus indicators
- Support keyboard activation

### Forms
- Clear labels and error states
- Accessible validation messages
- Proper ARIA attributes
- Logical tab order

### Navigation
- Semantic HTML structure
- Skip links for accessibility
- Clear visual hierarchy
- Responsive behavior

## Implementation Rules

### Do
- Use semantic tokens over raw values
- Preserve visual hierarchy
- Keep interaction states explicit
- Maintain consistent spacing rhythm

### Don't
- Avoid low contrast text combinations
- Avoid inconsistent spacing patterns
- Avoid ambiguous labels or instructions
- Skip accessibility considerations
